[Features]
1. Окно Filest:
  +1. Окно может ресайзиться
  +2. Добавлять новые файлсеты
  +3. Вывод списка файлов
  +4. Имя для fileset генерируется автоматически;
  +5. Возможность добавлять/удалять файлы;

    +1. Окно SearchView занимает всё отводимое ей место;
    +2. Строка фильтра во всю ширину вьюхи
    +3. Окно с результатами поиска во всю площадь вьюхи
  4. Базовый функционал поиска на основе данных filest
    1. Передавать имя fileset
    2. Передавать filter
  5. Вывод в TextArea
  6. Моноширинный шрифт
